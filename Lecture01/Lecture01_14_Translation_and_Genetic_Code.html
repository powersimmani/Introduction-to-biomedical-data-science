tRNA Structure and Function
• Cloverleaf secondary structure
• Anticodon loop (3 bases)
• Amino acid attachment site (3' end)
• Wobble base pairing at 3rd position
Genetic Code Properties
64 Codons: 61 code for amino acids, 3 stop codons (UAA, UAG, UGA)
Start Codon: AUG (codes for Methionine)
Redundancy: Multiple codons per amino acid (degenerate)
Universal: Nearly identical across all organisms
