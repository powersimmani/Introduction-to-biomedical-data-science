Hands-on: PyMOL Molecular Visualization
PyMOL: Open-Source Molecular Visualization Tool
Industry-standard software for protein structure analysis
1
Loading PDB Structures
fetch 1UBQ
• Direct download from PDB
• File → Open (local files)
• Example: Ubiquitin (1UBQ)
2
Rendering Options
• Cartoon (secondary structure)
• Surface (molecular surface)
• Sticks (chemical bonds)
• Spheres (space-filling)
• Ribbon (backbone)
3
Measuring Distances
distance name1, name2
• Click atoms with Ctrl
• Hydrogen bonds
• Active site measurements
4
Creating Animations
• mset 1 x60
• util.mroll()
• Save as movie
• Presentation mode
Useful PyMOL Commands
color blue, chain A - Color by chain
show surface - Display surface
hide everything - Clear display
select ligand, organic - Select ligands
zoom ligand - Focus on selection
ray 1200, 800 - High-quality rendering
Publication Figures: PyMOL produces publication-quality images; widely used in research papers
