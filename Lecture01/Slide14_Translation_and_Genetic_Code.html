Translation and Genetic Code
Ribosome Structure
Prokaryotes: 70S (50S + 30S)
Eukaryotes: 80S (60S + 40S)
rRNA + proteins
Three sites: A, P, E
tRNA Structure
Cloverleaf secondary structure
L-shaped tertiary structure
Anticodon pairs with mRNA codon
Amino acid attached at 3' end (CCA)
Genetic Code
64 codons (61 amino acids + 3 stop)
Degenerate: multiple codons per AA
Universal (with rare exceptions)
Start: AUG (Met), Stop: UAA, UAG, UGA
Translation Steps
Initiation: ribosome assembly at start codon
Elongation: peptide bond formation
Termination: release factors at stop codon
Speed: ~20 AA/second in eukaryotes
